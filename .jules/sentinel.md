## 2024-05-18 - [Fix Webhook Secret Leak in Next.js API Route]\n**Vulnerability:** Found a hardcoded error message in `app/api/webhooks/clerk/route.ts` that exposed internal server configuration details and environment variable names (`CLERK_WEBHOOK_SECRET`) when the secret was missing. Uncaught exceptions can also leak stack traces.\n**Learning:** In Next.js API routes, `throw new Error` with sensitive information might expose internals in a production environment if error boundaries or frameworks leak it, or simply give attackers unnecessary information about the internal app structure.\n**Prevention:** Catch missing configuration errors gracefully. Use `console.error` to log generic warnings on the server side and return secure, generic HTTP 500 Responses instead of throwing hard errors that contain sensitive strings.
