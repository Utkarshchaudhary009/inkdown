## 2023-10-27 - Virtuoso itemContent Inline Object Bottleneck
**Learning:** In `components/MarkdownRenderer.tsx`, the `Virtuoso` virtualization library's `itemContent` render prop was passing an inline `components={{ h1: ..., p: ... }}` object to `Streamdown`. Because `Virtuoso` aggressively renders and recycles DOM elements during fast scrolling, creating a brand new object reference on every render caused React to completely unmount and remount all heading/paragraph elements within the chunks, leading to DOM thrashing and jank.
**Action:** Always extract static component mapping objects or configurations passed to heavy render loops (like virtualized lists) into stable constants outside the component definition to prevent unnecessary React reconciliation and DOM re-creation.
